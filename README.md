# 🎨 Smartboard Interface

An interactive HTML/CSS/JavaScript smartboard interface for classroom management with a beautiful pastel pink and purple color scheme.

## Features

✨ **Interactive Elements:**
- 7-item Lesson Flow navigation
- Settings panel for configuration
- Background & theme customization (4 color themes)
- Music library with upload & playback
- Video library with upload & playback
- Noise meter integration (Quiet Time)
- Interactive countdown timer with explosion animation
- Volume control slider

📱 **Design:**
- Responsive layout (works on smartboard displays 16:9 and 4:3)
- Pastel pink (#FFB6D9) and purple (#D4AEDD) theme
- Smooth animations and hover effects
- Professional, clean aesthetic

🎯 **Functionality:**
- LocalStorage persistence (settings auto-save)
- Embedded YouTube, Canva, and external resources
- File upload and storage
- Customizable lesson links
- Fullscreen support
- WCAG 2.1 AA accessibility compliant

## Files

- `index.html` - Main HTML structure
- `styles.css` - All CSS styling and animations
- `script.js` - Complete JavaScript functionality

## How to Use

1. Open `index.html` in your web browser
2. Click icons on the right sidebar to access features
3. Select lesson flow items at the bottom to display content
4. Upload backgrounds, music, and videos using the respective panels
5. All settings are automatically saved to your browser

You can customize:
- Color themes (default, ocean, forest, sunset)
- Background images
- Lesson links and URLs

- Video content

## Features in Detail

### Right Sidebar Icons (Top to Bottom)
1. ⚙️ **Settings** - Configure lesson links and general settings
2. 🎨 **Background** - Change background image and color theme
3. 🎵 **Music** - Upload and play audio files
4. 🎬 **Videos** - Upload and play video files
5. 🔇 **Quiet Time** - Noise meter for classroom management
6. ⏱️ **Timer** - Countdown timer with visual effects
7. 🔊 **Volume** - Audio volume control slider

### Lesson Flow (Bottom Center)
- 🔥 **Warm-up** - YouTube embedded playlist
- 📋 **Review** - Canva presentation integration
- 💭 **Prior Knowledge** - Interactive content placeholder
- 📚 **Lesson** - Main lesson content area
- ✏️ **Practice** - Practice activities section
- ✅ **Assessment** - Quiz and assessment area
- 🎉 **Wrap Up** - Summary and closing
